## CILeite

O Centro de Inteligência do Leite desenvolve estudos estratégicos sobre a cadeia produtiva do leite, com foco em competitividade, inteligência de mercado, tendências e análise econômica dos sistemas de produção. 

Esta é uma reformulação de design proposta para a nova versão do site. A solução de design foi criada através da plataforma Figma.
